# 픽셀 슬라임 클리커 🟢

귀여운 도트 그래픽 클리커 게임. 슬라임을 클릭해서 코인을 모으고, 동물 친구들을 고용하고, 방을 꾸미고, 환생을 거듭하며 슬라임을 전설의 갓 슬라임까지 키우세요.

## 실행 방법

`index.html`을 브라우저에서 열면 끝. 빌드 도구나 서버는 필요 없습니다.
(픽셀 폰트는 CDN에서 불러오므로 오프라인에서는 폴백 폰트로 표시됩니다.)

**PWA**: http(s)로 서빙하면 홈 화면에 설치할 수 있고, 서비스워커가 시트·폰트를 캐시해 오프라인에서도 동작합니다. 휴대폰 설치는 GitHub Pages 등으로 배포 후 "홈 화면에 추가"를 사용하세요.

## 게임 요소

### 기본 루프
- **클릭** — 슬라임을 누르면 코인 획득. 찌그러짐 + 파티클 + 효과음. Space/Enter 키도 가능
- **퍼펙트 말랑** — 3% 확률로 클릭 보상 x5 잭팟
- **콤보** — 빠르게 연타하면 배율 증가 (병아리 시너지로 강화 가능)
- **친구들** — 병아리부터 코인 행성까지 8종 자동 생산 유닛. x1/x10/MAX 구매, 회수 시간 표시
- **강화** — 클릭 파워·생산량 업그레이드 11종

### 친구 시너지
각 친구는 보유 수(10/25/50/100마리)에 따라 역할 보너스를 해금합니다.

| 친구 | 역할 |
| --- | --- |
| 🐤 병아리 | 클릭·콤보 강화, 100마리부터 주기적 클릭 x2 축제 |
| 🐱 고양이 | 오프라인 보상·꿈 보너스 |
| 🐶 강아지 | 황금별 등장·보상 강화 |
| 🐰 로켓 토끼 | 미니 별똥별·할인 이벤트 강화 |
| 🦄 유니콘 | 장식·도감·성향 보너스 |
| 🐲 아기 드래곤 | 광란 시간·배율·보너스 강화 |
| 🐋 우주 고래 | 방치·오프라인 장기 성장 |
| 🪐 코인 행성 | 전체 경제·환생 영혼 (소프트캡 적용) |

상점의 각 친구 항목에 "다음 시너지"가 표시되고, 방 탭의 친구 도감에서 해금 현황과 최고 보유 기록을 볼 수 있습니다.

### 진화 축복 (로그라이트)
진화할 때마다 슬라임이 **축복 3개 중 1개**를 고릅니다 (이번 생 한정, 환생 시 초기화).
대부분 가산형 보너스 16종이며 현재 성향에 맞는 축복이 더 자주 등장합니다.
한 번 받아본 축복은 도감에 기록됩니다. 매 환생 런이 조금씩 다른 빌드가 됩니다.

### 도전 런 — 이상한 이번 생
환생 2회부터 제약을 안고 환생하는 도전 런 4종이 열립니다:
**침묵의 손**(클릭 수입 0, 병아리 3마리 동행) / **외길 목장**(친구 한 종만) /
**별빛 사냥꾼**(생산 5%, 별 보상 3배) / **서두르는 생**(15분 제한).
환생선에 도달하면 완수 — 별가루 큰 보상. 실패하거나 내려놓아도 잃는 것이 없습니다.

### 진화와 성향
- **진화 5단계** — 누적 코인에 따라 초록 → 바다 → 딸기 → 황금 킹 → 갓 슬라임. 진행바로 다음 진화까지 표시
- **성향** — 이번 생의 플레이 방식에 따라 근육/목장/별빛/꿈꾸는/윤회 성향이 붙고 눈동자 색과 작은 픽셀 흔적이 남음

### 환생 (프레스티지)
- 누적 10억 코인부터 **젤리 환생** 가능. 영혼 = √(누적/6천만)
- 수입 보너스는 **총 획득 영혼** 기준 (+5%/개) — 특성을 사도 줄지 않음
- 보유 영혼으로 영구 특성 6종 구매 (클릭, 생산, 황금별, 오프라인, 콤보, 진화 요구치)
- 환생 시 코인·친구·강화·이번 생 통계만 초기화. 업적·장식·도감·기록·영혼은 유지

### 황금별 이벤트 (5종)
광란의 시간(x7~x8) / 코인 보너스 / 친구 파티(생산 x5) / 별의 할인(50%) / 미니 별똥별

### 먹구름 손님
가끔 먹구름 슬라임이 찾아옵니다. 두드리면 10초 챌린지 시작 — 성공하면 큰 코인과 별가루,
**실패해도 위로 간식을 두고 가고, 무시해도 아무 손해가 없습니다.** 달래준 횟수는 앨범에 기록돼요.

### 슬라임의 방
- **살아있는 친구들** — 친구들이 2프레임 애니메이션으로 역할별 행동을 해요 (병아리는 쪼아먹기, 고양이는 낮잠, 강아지는 달리기, 토끼는 점프, 고래는 하늘 횡단…). 장식을 사면 좋아하는 자리로 이사
- **말랑 뉴스** — 해금 상태에 따라 달라지는 한 줄 소식이 가끔 흘러요
- **장식 6종** — 화면에 실제 배치되는 영구 보너스 (클릭·생산·황금별 체류). 환생해도 유지
- **도감** — 도달한 진화 단계, 경험한 성향, 만난 친구를 기록
- **별의 기록** — 진화·환생·성향 발견 마일스톤마다 해금되는 짧은 이야기
- **슬라임 앨범** — 함께한 날, 쓰다듬은 횟수, 최애 친구 같은 평생의 기억

### 슬라임의 소원과 별가루
- **소원** — 항상 3개의 작은 목표가 활성화 (클릭/콤보/황금별/고용/코인 모으기 등). 이루면 **별가루** 획득
- **별가루 꾸미기** — 오라 색, 배경 테마(새벽/심해/오로라), 말풍선 스킨, 클릭 파티클(하트/별/음표). 전부 표현 전용이라 밸런스에 영향 없음

### 기타
- **업적 21종** — 1개당 전체 수입 +1%
- **저장** — localStorage 자동 저장 + 세이브 버전 마이그레이션. 설정(⚙️)에서 세이브 코드 내보내기/가져오기 가능. 오프라인 보상 (기본 최대 8시간, 시너지로 확장)
- **슬라임 대사** — 성향 따라 달라지는 말풍선

## 기술

- 바닐라 HTML/CSS/JS 단일 `index.html`
- 슬라임·코인·별은 캔버스에 픽셀 단위로 직접 렌더링 (`image-rendering: pixelated`)
- 친구·아이템·업적 아이콘은 `assets/generated/sprites/`의 AI 생성 스프라이트 아틀라스 (스프라이트가 없는 항목은 이모지 폴백)
- 효과음은 WebAudio API로 생성
- 경제 루프는 rAF와 별도의 1초 인터벌로 구동 — 백그라운드 탭에서도 진행
- 모바일 레이아웃 지원

## 에셋

현재 친구, 아이템/특성, 업적, 장식 스프라이트 아틀라스를 사용합니다.
