# 픽셀 슬라임 클리커 🟢

귀여운 도트 그래픽 클리커 게임. 슬라임을 클릭해서 코인을 모으고, 동물 친구들을 고용해 자동으로 생산하고, 슬라임을 전설의 갓 슬라임까지 진화시키세요!

## 실행 방법

`index.html`을 브라우저에서 열면 끝. 빌드도, 서버도, 인터넷도 필요 없습니다.

## 게임 요소

- **클릭** — 슬라임을 누르면 코인 획득. 찌그러짐 애니메이션 + 파티클 + 효과음
- **콤보** — 빠르게 연타하면 최대 x2 배율
- **친구들** — 병아리부터 코인 행성까지 8종의 자동 생산 유닛
- **강화** — 클릭 파워, 생산량 배율 등 11종 업그레이드
- **진화** — 누적 코인에 따라 슬라임이 5단계 진화 (초록 → 바다 → 딸기 → 황금 킹 → 갓 슬라임)
- **황금별** — 랜덤 등장. 클릭하면 20초간 x7 광란 타임 또는 코인 보너스
- **저장** — localStorage 자동 저장 + 오프라인 보상 (최대 8시간)

## 기술

- 바닐라 HTML/CSS/JS 단일 파일
- 모든 그래픽은 캔버스에 픽셀 단위로 직접 렌더링 (`image-rendering: pixelated`)
- 효과음은 WebAudio API로 생성 (에셋 파일 없음)
- 모바일 레이아웃 지원
