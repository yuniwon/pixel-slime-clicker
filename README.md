# 픽셀 슬라임 클리커 🟢

귀여운 도트 그래픽 클리커 게임. 슬라임을 클릭해서 코인을 모으고, 동물 친구들을 고용하고, 방을 꾸미고, 환생을 거듭하며 슬라임을 전설의 갓 슬라임까지 키우세요.

**▶ 바로 플레이**: https://yuniwon.github.io/pixel-slime-clicker/

## 실행 방법

`index.html`을 브라우저에서 열면 끝. 빌드 도구나 서버는 필요 없습니다.
(픽셀 폰트는 CDN에서 불러오므로 오프라인에서는 폴백 폰트로 표시됩니다.)

**PWA**: http(s)로 서빙하면 홈 화면에 설치할 수 있고, 서비스워커가 시트·폰트를 캐시해 오프라인에서도 동작합니다. 푸시할 때마다 GitHub Pages가 자동 재배포됩니다.

## 플레이테스트 방법

실제 사람의 첫 20분 재미를 보려면 [플레이테스트 가이드](docs/PLAYTEST-GUIDE.md)를 사용하세요. 테스트 후 설정(⚙️)에서 세이브 코드를 내보내고, 콘솔의 `telemetryDump()` 결과와 5점 설문, "또 할래?" 답변을 함께 모읍니다. 텔레메트리는 로컬 세이브에만 저장되며 외부로 전송되지 않습니다.

## 게임 요소

### 기본 루프 — 손맛
- **클릭** — 슬라임을 누르면 코인 획득. 찌그러짐 + 파티클 + 효과음. Space/Enter 키도 가능
- **음계 클릭** — 클릭음이 BGM과 같은 C 펜타토닉으로 양자화된 3레이어 사운드 (노이즈 타격감 + 음계 + 10콤보마다 옥타브 반짝임). 연타가 그대로 연주가 됩니다. 안드로이드는 햅틱 진동 (설정 📳)
- **퍼펙트 말랑** — 3% 확률로 클릭 보상 x5 잭팟
- **콤보 열기 4단계** — 말랑→따끈→후끈→불타는 말랑. 경계를 넘는 클릭엔 승급 버스트 (히트스톱 + 색 링 + 아르페지오), 오라가 슬라임 주변에 차오릅니다
- **친구들 10종** — 병아리·고양이·강아지·로켓 토끼·유니콘·아기 드래곤·우주 고래·코인 행성·별의 정령·시간 거북. x1/x10/MAX 구매, 회수 시간 표시
- **강화 25종** — 클릭 파워·생산량·퍼펙트·콤보 업그레이드

### 매일의 의식 (데일리 루프)
- **오늘의 첫 말랑** — 하루의 첫 클릭은 보장된 잭팟 (큰 코인 + 별가루 + 따끈 콤보 시작 + 아침 인사). 놓쳐도 페널티 없음
- **황금 소원** — 소원 탭 맨 위 금테 카드. 날짜 시드로 매일 1개 교체, 목표는 일반의 ~3배 (손님 연동형 포함), 보상 별가루 5
- **젤리 결정** — 무대의 화단에서 결정이 실시간 22시간 숙성 (오프라인 진행). 20시간부터 조기 수확 도박 가능, 26시간엔 자동 수확이라 놓쳐도 손해 없음. 하루 1번 물 주기로 2시간 단축. 수확한 결정을 친구에게 먹이면 **영구 생산 +1%/레벨** — 환생에도 유지되는 메타 자원

### 친구 시너지
각 친구는 보유 수(10/25/50/100마리)에 따라 역할 보너스를 해금합니다.

| 친구 | 역할 |
| --- | --- |
| 🐤 병아리 | 클릭·콤보 강화, 100마리부터 주기적 클릭 x2 축제 |
| 🐱 고양이 | 오프라인 보상·꿈 보너스 |
| 🐶 강아지 | 황금별 등장·보상 강화 |
| 🐰 로켓 토끼 | 미니 별똥별·할인 이벤트 강화 |
| 🦄 유니콘 | 장식·도감·성향 보너스 |
| 🐲 아기 드래곤 | 광란 시간·배율·보너스 강화 |
| 🐋 우주 고래 | 방치·오프라인 장기 성장 |
| 🪐 코인 행성 | 전체 경제·환생 영혼 |
| 🧚 별의 정령 | 행운·별가루 |
| 🐢 시간 거북 | 시간·갈증 곡선 완화 |

### 슬라임 레벨과 진화 축복 (로그라이트)
- **레벨** — 누적 코인으로 레벨업. 레벨당 수입 +2%, Lv13+부터 갈증 곡선 (요구 배율 점증)
- **축복 3택 1** — 3레벨마다 선택. **41종 × 4등급** (일반·스택가능 / 희귀 / 영웅 / 전설). 성향에 맞는 축복이 3배 잘 나오고, **행운**이 상위 등급 확률을 올립니다 (소프트캡 30)
- **리롤·스킵** — 별가루로 리롤 (기본 3, 리롤마다 +2), 패스하면 별가루 +1 (환생 1회부터)
- 전설 축복은 게임 체인저: 쌍둥이 별, 영원한 박자(콤보 무한), 퍼펙트 연쇄, 잭팟

### 진화와 성향
- **진화 5단계** — 초록 → 바다 → 딸기 → 황금 킹 → 갓 슬라임 (Lv1/5/10/15/20). 본체는 AI 생성 스프라이트 (2프레임 idle), 다음 진화 실루엣 게이지 표시
- **성향** — 이번 생의 플레이 방식에 따라 근육/목장/별빛/꿈꾸는 성향이 붙고, 환생을 거듭하면 윤회의 고리가 생깁니다. 성향은 눈동자 색·오라 글로우로 표현

### 환생 (프레스티지)
- 누적 10억 코인부터 **젤리 환생**. 영혼 = 깊이의 로그 함수 (floor(1 + 3·log₁₀(E/6e7)^2.2)) — 깊이 들어갈수록 가속
- 수입 보너스는 **총 획득 영혼** 기준 소프트캡 (+5%×100 / +2%×400 / +0.5%) — 특성을 사도 줄지 않음
- 보유 영혼으로 영구 특성 10종 구매 (클릭·생산·황금별·오프라인·콤보·진화·행운·고비용 4종)
- **깊은 층 시작** — 환생 1.5회당 시작 레벨 +1 (최대 Lv15), 건너뛴 구간의 축복 일괄 선택
- 환생 시 코인·친구·강화·이번 생 통계만 초기화. 업적·장식·도감·영혼·젤리 결정은 유지

### 도전 런 — 이상한 이번 생 (환생 2회부터)
**침묵의 손**(클릭 수입 0, 병아리 3마리 동행) / **외길 목장**(친구 한 종만) /
**별빛 사냥꾼**(생산 5%, 별 보상 3배) / **서두르는 생**(15분 제한).
환생선에 도달하면 완수 — 별가루 큰 보상. 실패하거나 내려놓아도 잃는 것이 없습니다.

### 하늘의 손님들
- **황금별** (5종 룰렛) — 광란의 시간(x7~x8) / 코인 보너스 / 친구 파티(생산 x5) / 별의 할인(50%) / 미니 별똥별
- **먹구름 손님** — 두드리면 10초 연타 챌린지. 성공하면 큰 코인과 별가루, **실패해도 위로 간식을 주고, 무시해도 손해 없음**
- **길 잃은 별똥이** — 천천히 이끌어 집까지 데려다주는 손님. 보상 + 별가루

### 슬라임의 방
- **살아있는 친구들** — 친구들이 2프레임 애니메이션으로 역할별 행동 (쪼기·낮잠·달리기·점프·하늘 횡단…). 장식을 사면 좋아하는 자리로 이사
- **말랑 뉴스** — 해금 상태에 따라 달라지는 한 줄 소식 티커
- **기록의 선반** — 최근 업적 트로피가 무대 하단에 물리적으로 켜지는 공간
- **장식 6종** — 화면에 실제 배치되는 영구 보너스. 환생해도 유지
- **도감 / 별의 기록 / 슬라임 앨범** — 진화·성향·친구 기록, 마일스톤 스토리 12편, 평생의 기억

### 슬라임의 소원과 별가루
- **소원** — 항상 3개의 작은 목표 (클릭/콤보/황금별/고용/코인 등 7종). 이루면 **별가루**
- **별가루 꾸미기 11종** — 오라·배경(새벽/심해/오로라)·말풍선·클릭 파티클. 전부 표현 전용이라 밸런스 영향 없음

### 기타
- **업적 37종** — 1개당 전체 수입 +1%
- **BGM** — WebAudio 프로시저럴 칩튠 자장가 (C장조 펜타토닉 72bpm, 광란 시 한 옥타브 위). 효과음·음악·진동 별도 토글
- **저장** — localStorage 자동 저장 + 세이브 버전 마이그레이션. 설정(⚙️)에서 세이브 코드 내보내기/가져오기. 오프라인 보상 (기본 최대 8시간, 특성·시너지로 확장)
- **우주 씨앗** — 갓 슬라임·환생 3회·성향/기록/방의 흔적을 모으면 열리는 중간 엔딩

## 기술

- 바닐라 HTML/CSS/JS 단일 `index.html` (~5,500줄), 외부 라이브러리 0
- 슬라임 본체는 AI 생성 아틀라스를 캔버스 CSS 배경으로 렌더 + 성향 오버레이는 캔버스 픽셀 (아틀라스 로드 실패 시 16×17 수제 픽셀 렌더로 자동 폴백)
- 아이콘·친구·손님은 `assets/generated/sprites/`의 스프라이트 아틀라스, 없는 항목은 이모지 폴백 — [에셋 파이프라인 규약](docs/ASSET-PIPELINE.md)
- 효과음·BGM은 WebAudio API로 실시간 생성 (오디오 에셋 0)
- 경제 루프는 rAF + 1초 인터벌 이중 구동 — 백그라운드 탭에서도 진행
- 모바일: 100dvh 고정 분할, safe-area, iOS 더블탭 줌 차단 (touch-action + touchend 가드), 해상도 비례 UI (vw clamp)
- 메타데이터: OG/Twitter 카드, canonical — 링크 공유 시 카드 렌더

## 내장 개발 도구 (콘솔)

- `simRun({hours, clicksPerSec, fromCurrent})` — 봇 플레이 시뮬레이션 (밸런스 검증)
- `launchCheck()` — 출시 판정 하니스 54검사 (안정성/온보딩/페이싱/성능/준수 가중 평가) — [출시 기준](docs/LAUNCH-CRITERIA.md)
- `telemetryDump()` — 로컬 재미 텔레메트리 요약 (외부 전송 없음) — [플레이테스트 가이드](docs/PLAYTEST-GUIDE.md)
