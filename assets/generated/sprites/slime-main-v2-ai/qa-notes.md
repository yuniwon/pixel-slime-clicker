# slime-main-v2-ai QA

Generated to replace the older default slime atlas with the newer baked-trait pixel style. Five default stages, 2-frame idle each. No detached star sparkles, hats, headbands, or nightcaps. Sprites are scaled inside 768px runtime cells to match the larger baked-trait read size. Magenta fringe was stripped with wider tolerance except the strawberry row, which uses a narrow key to preserve pink body pixels. Muscle variants remain in the baked trait atlases and are connected separately.
